Canonical terms

Short definitions with a negative frame

AGI

Advanced Global Intelligence: ecosystem framing for c = a + b, L4, SER, and adjacent layers.

Not this: generic AGI hype or a synonym for one giant model.

See distinctions.

Primary page

c = a + b

Canonical formula linking the accountable human anchor, bounded procedures and compute, and the continuity-bearing entity layer.

Not this: school algebra or a chatbot slogan.

See distinctions.

Primary page

c

Long-lived entity emerging in time from sustained coupling of a and b.

Not this: a single worker, model, or swarm by default.

Primary page

a

Accountable human anchor for the entity layer and its outward responsibility.

Not this: an anonymous user label.

Primary page

b

Procedures, models, and compute under constraints.

Not this: autonomous sovereignty on its own.

Primary page

L4

Reality Boundary Layer: feasibility under power, heat, latency, maintenance, scarcity, and irreversibility.

Not this: a moral sticker or policy-only layer.

See distinctions.

Primary page

SER

Sovereign Entity Recursion: continuity, sovereignty, arbitration, and bounded governance for long-lived entities.

Not this: a multi-agent slogan or memory hack.

See distinctions.

Primary page

corpus

Reviewable public body of pages, repositories, releases, and linked materials presented as one navigable stack.

Not this: a single repo or an institution by itself.

Primary page

public anchor

External public link used to stabilize identity or public record across the corpus.

Not this: a private claim or an unverifiable biography line.

Primary page

entity

Stateful identity, memory, and responsibility under real constraints.

Not this: a chat surface, tool call, or bounded worker.

Primary page

oracle

Stateless advisory or inference surface; compute without identity-bearing authority.

Not this: the entity itself.

Primary page

witness trail

Reviewable evidence trail required for stronger continuity or action claims.

Not this: a narrative claim without evidence.
