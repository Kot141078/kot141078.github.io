# Diary Assets

Use this directory for diary images that belong to imported public posts.

Recommended layout:

- `assets/diary/<slug>/cover.ext`
- `assets/diary/<slug>/image-02.ext`
- `assets/diary/<slug>/image-03.ext`

Rules:

- Keep original filenames only if they stay readable.
- Prefer one folder per post slug.
- Prefer one clear cover image only when the entry actually has a primary image.
- Use gallery-style additional files only when the source batch really contains multiple publishable images.
- Do not remove or crop source images silently.
- If an image is omitted from the site version, note that in the import pass report.
