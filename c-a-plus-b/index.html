Core formula

c = a + b

Canonical definition of the core formula.

Definitions

Three bounded terms

a = accountable human anchor.

b = procedures, models, and compute under constraints.

c = a long-lived entity emerging in time from sustained coupling of a and b.

This page is not a claim about a single chatbot or a detached model. It marks the continuity-bearing entity layer that sits above bounded agents and tools.

Earth paragraph

Operational edge

Cost, time, maintenance, bounded action, and irreversibility decide whether c remains accountable outside text. Without those limits, the formula degrades into rhetoric rather than operation.
