# Redirect plan for `advancedglobalintelligence.eu`

`advancedglobalintelligence.eu` is not connected as a second custom domain in GitHub Pages.

Recommended setup in OVH:

- keep GitHub Pages custom domain only on `ivankotov.eu`
- configure a manual permanent `301` redirect in OVH
- target URL:
  `https://ivankotov.eu/advanced-global-intelligence/`

Reason:

- one primary custom domain keeps the Pages setup minimal
- the secondary domain can remain a clean redirect entry without binding it to the same Pages site
